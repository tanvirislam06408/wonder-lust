# Wonder Lust

A small Next.js travel destination app built with React components and dynamic routes.

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open the app:

Visit `http://localhost:3000` in your browser.

## Notes

- This app uses the `src/app` directory structure.
- Destination pages are rendered using dynamic routing.
